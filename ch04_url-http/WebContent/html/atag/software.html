Software

A term made up of the opposite of hardware,programing  a physical entity. It may be interpreted in the same way as a computer program, but it refers to the entire program, including its own manual, its rules and procedures, and manuals and indexes that document them.
It has also distinguished between system software, which is essential for running a computer system, and application programs, which are specific to users on the user side. Currently, it is called software including both systems. Software development is treated as a synonym.
[Note] Intelligent creation that includes programs, procedures, rules, and related documentation to make the data processing system functional.
